---
title: "HTTP và REST API với JavaScript"
date: 2025-01-06
---

HTTP là giao thức nền tảng của web.
Trong JavaScript, HTTP thường được sử dụng để xây dựng REST API.

REST API cho phép các hệ thống khác nhau giao tiếp
thông qua các phương thức như GET, POST, PUT và DELETE.

Node.js kết hợp với các framework như Express.js
giúp việc xây dựng API trở nên nhanh chóng và rõ ràng.

REST API đóng vai trò quan trọng trong các hệ thống client-server hiện đại.
